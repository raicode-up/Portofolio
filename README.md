# With love, Rai. — Personal Portfolio Website

Welcome to the official repository of my personal portfolio website! Website ini dirancang sebagai representasi digital dari profil dan saya di bidang Sistem Informasi.

## ✨ Tentang Website
Website portofolio ini milik **Nabila Raisiya Zahrani (Rai)**, mahasiswi Sistem Informasi Universitas Brawijaya. Mengusung tema modern dan clean dengan sentuhan estetika *Glassmorphism* serta palet warna pastel yang terinspirasi dari kombinasi warna *Rose Quartz* & *Serenity* (serta dedikasi kecil untuk grup *SEVENTEEN*).

## Fitur Utama
- **Desain Responsif & Fleksibel:** Tampilan web otomatis menyesuaikan dengan sempurna di mobile, tablet, hingga layar desktop lebar.
- **Glassmorphism Navbar:** Efek blur transparan (`backdrop-blur`) pada navbar memberikan kesan modern yang menyatu dengan latar belakang saat di-scroll.
- **Efek Animasi On-Scroll:** Halaman dilengkapi efek animasi transaksi memudar dan muncul ke atas yang halus menggunakan *Vanilla JavaScript Intersection Observer API*.
- **Hamburger Menu Mobile:** Menu navigasi ramah pengguna yang interaktif dan responsif pada perangkat berlayar kecil.
- **Integrasi Brand Icons:** Menggunakan Lucide Icons v0.344.0 yang menampilkan ikon media sosial.

## Tech Stack
- **Struktur Halaman:** HTML5
- **Framework Gaya (Styling):** Tailwind CSS (melalui CDN resmi)
- **Desain Tambahan:** Custom CSS (`style.css`) dengan variabel gradasi warna kustom, animasi, dan efek kaca.
- **Ikonografi:** Lucide Icons v0.344.0
- **Tipografi:** Google Fonts (Font utama: *Inter*, Font judul: *Poppins*)
- **Interaktivitas & Logika:** Vanilla JavaScript
